PBEM Box is for manual PBEM workflows where turns are sent by email, cloud drive, chat, or another service.

![PBEM Box header with opponent management, turn import, and refresh controls.](../../../assets/screenshots/warhq-pbem-box.png)

## Import a received turn

Use **Import Turn** when you want to browse for a file manually.

1. Open PBEM Box from the Library toolbar.
2. Click **Import Turn**.
3. Choose the received `.bte` or `.cpf` file.
4. Select the destination game if WarHQ asks.
5. Keep the suggested backup option enabled when replacing an existing file.

## Drag and drop a received turn

You can also drag a received turn file directly onto WarHQ. This works from the Library and from PBEM Box.

![WarHQ showing the green PBEM turn drop target while a received turn is dragged over the app.](../../../assets/screenshots/warhq-pbem-drag-drop.png)

1. Save the received turn attachment somewhere easy to find, such as Downloads or your desktop.
2. Drag one supported PBEM turn file into WarHQ. WarHQ accepts `.bte` and `.cpf` files.
3. Wait for the green **Release to import this turn** overlay.
4. Release the mouse button.
5. Confirm the import options in the dialog that opens.

If WarHQ does not show the green overlay, check that you are dragging only one `.bte` or `.cpf` file.

## Prepare an outgoing turn

After you finish a PBEM turn, WarHQ can detect the updated save file and prepare an email draft. Review the recipient, subject, and attachment before sending.

## Organize opponents

Use opponent assignment when several files belong to the same PBEM match. This makes the PBEM list easier to scan and helps WarHQ group related files.
